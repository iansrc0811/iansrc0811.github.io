Mac 上設定多個 GitHub 帳號的 SSH Key

在同一台 Mac 上有多個 GitHub 帳號（例如工作用和個人用），push 時常常會遇到權限錯誤：

ERROR: Permission to xxx/repo.git denied to yyy.

這是因為 SSH agent 會優先送出第一把 key，GitHub 一認到帳號就不會再試其他 key 了。
